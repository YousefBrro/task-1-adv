# 📚 Smart Library Management System

An interactive web application for book management, designed to bridge the gap between complex logic and user interface (UI) using **TypeScript**.

---

## 🧠 OOP Principles Applied
This project focuses on clean, scalable code through the following Object-Oriented Programming concepts:

* **Abstraction:** Focused on essential properties (Title, Author, Status) within the `Book` class, hiding internal complexities from the user.
* **Encapsulation:** Protected book data using private properties (e.g., `#books`) to ensure data integrity, allowing modification only through specific methods like `addBook`.
* **Inheritance (Future-Ready):** Structured to easily extend functionality, such as creating a `DigitalBook` class that inherits from the base `Book` class.
* **Polymorphism:** Managed various book objects through a unified interface, allowing the system to process and render items correctly regardless of their specific type.

---

## 🛠️ Technologies Used
* **TypeScript:** Used for robust type safety and preventing runtime errors.
* **HTML5 & CSS3:** Developed with a clean, **Minimalist Dark Tech** aesthetic for a professional user experience.

---
